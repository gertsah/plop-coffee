# Plop — кофейня в Новороссийске

Лендинг кофейни **Plop** (Анапское шоссе, 41, Новороссийск).

Статический сайт: HTML / CSS / JS, без сборки.

## Локальный запуск

Перемотка видео (scroll-scrub) требует поддержки HTTP Range, поэтому используйте сервер с Range:

```bash
python serve.py 5577
```

Затем откройте http://localhost:5577

(Обычный `python -m http.server` не поддерживает перемотку видео.)

## Структура

- `index.html` — разметка
- `styles.css` — стили
- `script.js` — интерактив (sticky-hero, scroll-scrub видео, авто-лента отзывов, табы меню, бургер)
- `assets/` — медиа (логотип, видео, фото, QR, текстуры)
- `serve.py` — локальный сервер с поддержкой Range
- `cutout.py` — утилита вырезки фона у фото стакана
